# HR Attrition Analysis Dashboard using Excel

##  Project Overview

This project presents an HR analytics dashboard built using Microsoft Excel to analyze employee attrition trends. The dashboard helps identify key factors influencing employee turnover and supports data-driven HR decisions.

---

## Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning & Transformation
* Slicers and KPI Cards
* Data Visualization

---

## Dataset Description

The dataset includes:

* Employee details
* Department information
* Gender distribution
* Attrition records
* Monthly attrition data

---

##  Dashboard Features

###  Key Performance Indicators (KPIs)

* Total Employees: 150
* Active Employees: 138
* Total Attrition: 12
* Attrition Rate: 8%

---

###  Attrition by Department

* Compares attrition across departments:

  * Sales
  * IT
  * HR
  * Finance
  * Operations
* Helps identify departments with higher employee turnover

---

###  Attrition by Gender

* Visualizes attrition distribution between male and female employees
* Helps understand demographic impact on attrition

---

###  Monthly Attrition Trend

* Tracks attrition over time
* Helps identify patterns and stability in employee turnover

---

##  Key Insights

* Sales department has the highest attrition
* Male employees contribute more to attrition compared to female employees
* Attrition trend remains stable without significant spikes
* Focused retention strategies are needed for high-risk departments
